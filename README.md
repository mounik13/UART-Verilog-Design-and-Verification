# UART Verilog Design and Verification

## Overview
This project implements a UART (Universal Asynchronous Receiver Transmitter) using Verilog. It includes a Transmitter (TX), Receiver (RX), and a baud rate generator.

## Features
- UART TX module
- UART RX module
- Baud rate generator
- FSM-based design
- Testbench for verification

## Tools Used
- Verilog HDL
- Vivado Simulator

## Simulation
Waveforms verified using Vivado.
